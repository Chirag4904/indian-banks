# Indian Banks 
Structured, centralized data, including logos for all the Indian Banks, collected from official sources.

## Current Status

## Data Points for Each Bank
- Name
- Logo
- Symbol / Small Logo
- Short Code
- IFSC Prefix
- Balance Check USSD Code
- Website
- Type : https://www.rbi.org.in/scripts/banklinks.aspx
